ls = list files and Directories 
example= ls 

ls -l= list files and Directories with detaild info
example = ls -l

ls -a= list the hidden files also 

Note : You can list files of another directory without changing
your current directory/location ie ls /Desktop

pwd=print working directory

cd= Navigate/change to specific directory
example: cd Downloads 

cd .. = move up one directory level
example= if current dir is /home/kali/Desktop the after
cd .. it will move up to /home/kali

cd ~ = gO to home directory

cd / = go to root directory

tree = Display the Directory structure as a Tree
