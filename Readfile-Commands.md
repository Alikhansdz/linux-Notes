cat = Dislay the entire file from start to end

example = cat Navigation-commands.sh

tac = same as cat but Displays in reverse order

more= Displays a file one page at a time (mainly scroll forward ) 

example= more file.txt 

less =Display a file one page at a time but with more Navigation
features(can scroll up and down etc )

example = less file.txt

head= show the first 10 lines by Default of the file

example= head file.txt , or if you want customize
(head -n N file.txt) where N is the number of lines you
 want to print

tail= show the last 10 lines by default of the file

example =same as head, just change the command to tail
