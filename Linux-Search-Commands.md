find = Search for files and directories in specified location based in crieteria such as 
name,type, size or permissions

example= find /home  -name "mypic.png" 

locate = Quickly finds files and directiories using pre-built database of file paths  

example= locate mypic.png

which = Show full path of Executable command

example= which sudo

whereis = locate binaries(executable), source code and manual pages of a command or program

example= whereis sudo

grep = Search for specific text or pattern in files

example = grep "pwd" /home/kali/linux-Notes/Linux-Navigation-Commands.md

egrep OR grep-E = More advance version of grep 

example= egrep "ls" /home/kali/linux-Notes/Linux-Navigation-Commands.md
